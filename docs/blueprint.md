# **App Name**: Formatastic

## Core Features:

- JSON Beautifier: Format and beautify JSON code for readability.
- JSON Parser: Parse JSON code and display it in a tree-like structure.
- XML Formatter: Format XML code for better readability.
- JS Beautifier: Format and beautify JavaScript code.
- Theme Switcher: Allow users to switch between different visual themes (e.g., light and dark mode).
- Format Suggestion: AI tool that suggests best-practice formats and layouts for code, enhancing readability and maintainability.

## Style Guidelines:

- Primary color: Deep Indigo (#6666FF), conveying professionalism.
- Background color: Light Gray (#E0E0EB) for a clean, neutral canvas that does not compete with formatted code.
- Accent color: Sky Blue (#8FB8FF), providing visual highlights that support readability without distracting.
- Body and headline font: 'Inter', sans-serif.
- Code font: 'Source Code Pro' for clear, legible code display.
- Use clean, simple icons for actions like format, parse, and download.
- Consistent 2-column layout with input on the left and formatted output on the right.
- Subtle, quick animations for actions like theme switching and formatting to give immediate feedback to user interaction.
